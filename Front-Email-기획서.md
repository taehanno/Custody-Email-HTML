## 4. 이메일 미발송 항목 (의도적 제외)

아래 이벤트는 플랫폼 내 알림(토스트/배너/To-do)으로 충분히 전달되거나 사용자 피로도 방지를 위해 이메일을 발송하지 않습니다.

| 지갑 생성 완료 | 플랫폼 내 카드 자동 등록으로 확인 가능 |
| 지갑 삭제 완료/반려 | 삭제 현황 대시보드 + To-do 카드로 충분 |
| OTP 재등록 | 본인이 직접 수행하는 액션이므로 불필요 |
| 비밀번호 변경 | 본인이 직접 수행하는 액션이므로 불필요 |
