§12-13-21-4-1 입금 거래 증빙 데이터 분기 규칙
거래 ID 접두어  →  DEP-YYYYMMDD-NNNNN (출금 INF- 와 구분)
송신 주소 유형 = Whitelist 등록 주소  →  "거래 구분: 정상 입금"
송신 주소 유형 = 미등록 주소 + Admin 승인  →  "거래 구분: 보류 후 승인" + 보류 후 승인 정보 박스 4행 추가
코인 = XRP  →  "Destination Tag" 행 추가, 블록 표기 "레저 인덱스"로 변경
코인 = XLM / EOS  →  "Memo" 행 추가 (값 또는 "—")
코인 ≠ XRP·XLM·EOS  →  Memo / Tag 행 미표시
입금 감지 ≠ 잔액 반영  →  두 일시 모두 표기 (1 confirm 기준 시점차)
입금에는 신청 ID·서명 이력·네트워크 수수료·총 출금액 항목 미표시
영문 발급  →  모든 항목 영문
