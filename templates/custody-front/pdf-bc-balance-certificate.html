§12-13-21-2 데이터 분기 규칙
보유 코인 > 20개  →  표를 다음 페이지로 분할, 헤더/푸터 반복
발급 범위 = 고객사 전체 지갑  →  모든 활성 지갑 자동 포함
발급 범위 = 특정 지갑 선택  →  선택 지갑만 + 발급 정보에 "선택 지갑: N개" 추가
잔액 = 0  →  표에서 제외
보류 상태 입금  →  제외 + ※ 표기
시세 조회 실패  →  평가금액 칸 "—" + ※ "시세 조회 불가 항목 있음"
영문 발급 (ENG)  →  모든 항목 영문 / 고객사명 KYB 영문명 우선
국·영문 병기  →  항목명 국문 / English, 본문 데이터는 단일 표기
