§12-13-21-4 출금 거래 증빙 데이터 분기 규칙
거래 ID 접두어  →  INF-YYYYMMDD-NNNNN (입금 DEP- 와 구분)
코인 = XRP  →  "Destination Tag" 행 추가 (값 또는 "—(미입력)")
코인 = XLM / EOS  →  "Memo" 행 추가 (값 또는 "—")
코인 ≠ XRP·XLM·EOS  →  Destination Tag / Memo 행 미표시
수신 유형 = 외부 주소 (Whitelist)  →  "Whitelist 이름" 행 추가
수신 유형 = 내부 지갑 이전  →  "거래 구분: 내부 지갑 이전" + "수신 지갑명" 행 추가, Whitelist 행 미표시
네트워크 수수료 = 가스 코인 별도 (예: ERC20 USDT)  →  출금 수량과 수수료를 코인 단위로 별도 표기
출금 중 (TXID 있음 / 1 confirm 미만)  →  상태: "출금 중", 블록 확인: "0 confirm — 전파 중"
메모 미입력  →  "메모" 행 미표시
출금에는 송신 주소·블록 높이·확정 수·입금 감지/잔액 반영 일시 항목 미표시
영문 발급  →  모든 항목 영문
