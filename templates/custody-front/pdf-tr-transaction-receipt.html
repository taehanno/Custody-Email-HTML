§12-13-21-4 데이터 분기 규칙
코인 = XRP  →  "Destination Tag" 행 추가 (값 또는 "—(미입력)")
코인 ≠ XRP  →  Destination Tag 행 미표시
출금 — 내부 지갑 이전  →  "수신 유형: 내부 지갑 이전" + "수신 지갑명" 추가
출금 — Whitelist  →  "수신 유형: Whitelist 등록 주소" + "Whitelist 이름" 추가
입금  →  신청 ID / 서명 이력 / 메모 미표시, 수수료 / 총 출금액 미표시
출금 중 (TXID 있음)  →  상태: "출금 중", 블록 확인: "0 confirm — 전파 중"
메모 미입력  →  "메모" 행 미표시
영문 발급  →  모든 항목 영문
