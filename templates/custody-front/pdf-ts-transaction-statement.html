§12-13-21-3 페이지 분할 및 데이터 분기 규칙
거래 ≤ 30건  →  1페이지 (요약+내역+직인 모두)
거래 31~80건  →  1p 요약+30건 → 2p 50건+직인
거래 81~130건  →  1p 요약+30건 → 2p 50건 → 3p 50건+직인
거래 0건  →  "조회 기간 내 거래 내역이 없습니다." 빈 행 표시. 직인 정상.
거래 > 10,000건  →  발급 자체 차단
XRP 거래 포함  →  Destination Tag는 본 표 미포함, TR 발급으로 별도 안내
출금 수수료  →  본 표에는 수신 금액만, 마지막 페이지 부록에 별도 표
